Pallet - First Steps

The simplest way to start pallet (but definitely not the nicest) is with a git checkout of the source and maven (see below). 

(require 'pallet.maven)
(require 'pallet.compute)
(require 'org.jclouds.compute)

(pallet.compute/supported-clouds) ; show supported clouds

;; Open a connection to the cloud api specified in ~/.m2/settings.xml
(def service
     (apply org.jclouds.compute/compute-service
      (concat (pallet.maven/credentials) [:log4j :enterprise :ssh])))

;; List all the nodes
(org.jclouds.compute/nodes service)

See also demo API code until it gets migrated into these docs.

Install

Pallet can be used as a jar file or from a source code checkout. It is available at Clojars, and can be used with maven, leiningen or any maven repository aware tool.

Source

Pallet is available on github. Pallet requires Clojure 1.2, and uses jclouds. 

git clone git://github.com/hugoduncan/pallet.git

You can start a REPL using maven. 

mvn clojure:repl

lein repl

Using leiningen or maven you can also start a swank session, and connect to it with Emacs, etc. 

mvn clojure:swank -Dmaven.test.skip=true

lein swank
