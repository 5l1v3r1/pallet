First Steps

The simplest way to start pallet is with a git checkout of the pallet-examples project.

Install

Pallet can be used as a jar file or from a source code checkout. Release versions are available at Sonatype, and can be used with maven, leiningen or any maven repository aware tool.

Lein/Cake

Include the following in your project.clj 

:dependencies [[org.cloudhoist/pallet "0.3.0"]
               [org.cloudhoist/pallet-crates-complete "0.3.0"]]
:repositories {"sonatype"
               "http://oss.sonatype.org/content/repositories/releases"
               "sonatype-snapshots"
               "http://oss.sonatype.org/content/repositories/snapshots"}

Maven

Include the following in your pom.xml. 

<dependencies>
  <dependency>
    <groupId>org.cloudhoist</groupId>
    <artifactId>pallet</artifactId>
    <version>0.4.0-SNAPSHOT</version>
  </dependency>
  <dependency>
    <groupId>org.cloudhoist</groupId>
    <artifactId>pallet</artifactId>
    <version>0.4.0-SNAPSHOT</version>
  </dependency>
<dependencies>

<repositories>
  <repository>
    <id>sonatype</id>
    <url>http://oss.sonatype.org/content/repositories/releases</url>
  </repository>
  <repository>
    <id>sonatype-snapshots</id>
    <url>http://oss.sonatype.org/content/repositories/snapshots</url>
  </repository>
</repositories>

Source

Pallet is available on github. Pallet requires Clojure 1.2, and uses jclouds. 

git clone git://github.com/hugoduncan/pallet.git
git clone git://github.com/hugoduncan/pallet-crates.git
