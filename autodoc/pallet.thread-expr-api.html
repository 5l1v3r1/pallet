API for pallet.thread-expr

by Unknown

Usage: 
(ns your-namespace
  (:require pallet.thread-expr))

Overview


Macros that can be used in an expression thread




Public Variables and Functions



  

  

  
apply->

  macro

  
Usage: (apply-> request f & args)


  
Apply in a threaded expression.
e.g.
   (-> 1
     (apply-> + [1 2 3]))
=> 7

  


  

  

  
apply-map->

  macro

  
Usage: (apply-map-> request f & args)


  
Apply in a threaded expression.
e.g.
   (-> :a
     (apply-map-> hash-map 1 {:b 2}))
=> {:a 1 :b 2}

  


  

  

  
for->

  macro

  
Usage: (for-> arg [value s & {:keys [let]}] & body)


  
Apply a thread expression to a sequence.
eg.
   (-> 1
     (for-> [x [1 2 3]]
       (+ x)))
=> 7

  


  

  

  
if->

  macro

  
Usage: (if-> arg condition form)
       (if-> arg condition form else-form)


  
An `if` form that can appear in a request thread
eg.
   (-> 1
     (if-> true
       (+ 1)
       (+ 2)))
=> 2

let-with-arg->

  macro

  
Usage: (let-with-arg-> arg arg-symbol binding & body)


  
A `let` form that can appear in a request thread, and assign the value of the
threaded arg.
eg.

     (let-with-arg-> val [a 1]
