API for pallet.execute

by Unknown

Usage: 
(ns your-namespace
  (:require pallet.execute))

Overview


Exectute commands.  At the moment the only available transport is ssh.




Public Variables and Functions



  

  

  
execute-ssh-cmds

  function

  
Usage: (execute-ssh-cmds server request user options)


  
Run cmds on a target.

  


  

  

  
local-checked-script

  macro

  
Usage: (local-checked-script msg & body)


  
Run a script on the local machine, setting up stevedore to produce the
correct target specific code.  The return code is checked.

local-sh-cmds

  function

  
Usage: (local-sh-cmds {:keys [commands root-path], :or {root-path /tmp/}, :as request})

  function

remote-sudo-cmd

  function

  
Usage: (remote-sudo-cmd server session sftp-channel user tmpfile command)


  
Execute remote command.
Copies `command` to `tmpfile` on the remote node using the `sftp-channel`
and executes the `tmpfile` as the specified `user`.
